# zadmin-php-adminpanel
Zadmin php admin panel is a starter kit for the people who is code with php. its have own installer, admin manager, etc 


__Features__
  *Easy installer*
  *Admin Manager*
  *Forget password*
  *Clean ui*
  *Documentation* 


__Requirements__
  *PHP 7.2 or higher
  *MySQLi
